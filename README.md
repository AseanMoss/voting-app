# Voting app (PyQt6)

## Description
This is a GUI voting application buiilt using PyQt6. Useres can vote for john or jane, reset votes, and results are saved to a CSV file.

## Features
vote for john or jane
Reset votes
Saves data to a CSV file
GUI built with Qt designer

## How to Run
Install PyQt6
py -m pip install PyQt6

Run the app
python main.py

## File Structure
models/ -> voting logic

services/ -> file handlng

gui/ -> UI files

data/ -> CSV file
